## Requisitos
Para poder correr el proyecto se necesitan tener:
- Python3
- pip3
- pipenv

## Dependencias
El proyecto utiliza distintas dependencias. Para poder usarlo hay que insatalarlas corriendo el siguiente comando:
```shell
pipenv install
```
